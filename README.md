# starwarrs
Starwars character list.

This is a Basic React + Typescript app that implements the Starwars api [SWAPI](https://swapi.dev/).

Stack:
 - ###### React
 - ###### TypeScript
 - ###### Zustand (State)
 - ###### Axios (REST)
 - ###### SCSS
 - ###### Vite
 
 To run in local clone the repo and run the below commands:
 ```
 yarn install
 yarn run dev
 ```
